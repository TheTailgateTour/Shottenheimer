# The Shottenheimer — KC Tailgate QR Menu

Files in this folder power a tiny GitHub Pages site:

- `index.html` — the menu
- `styles.css` — theme (KC red/gold)
- `qr.html` — print & download a QR code that links to the site root
- `poster.html` — letter-sized printable poster with a big QR and menu highlights

## Publish on GitHub Pages
1. Create a new GitHub repo (e.g., `shottenheimer`).
2. Upload these files to the repo root.
3. Settings → Pages → Source: **Deploy from a branch**. Branch: `main`, Folder: `/root`.
4. After a minute your site is live at: `https://<user>.github.io/<repo>/`.
5. Open `/qr.html` to print the QR, or `/poster.html` to print a 1‑page poster.
